Features

Simple, Utility First

Simple and easy, straight-forward class names that anyone can remember!

Moblie-first

Mobile First Library that turns your design into something that matters.

Library Not Framework

Aroma is not a framework, its a library or "framework extension". It adds minimalistic design and some components your css framework may haven't includeded.

Useful Components, yet no javascript

Useful components like cards, navbar and flex-boxes without javascript at all.

Various components and re-designed elements

Article, p, button, cards, navbar, footer, progress and many others
NOTE: Not all components are yet available in this alpha stage.

Free and open source

Remix on glitch, or get from GitHub. (GitHub isn't available yet!).
